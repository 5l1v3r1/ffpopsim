integerify Function Reference¶

integerify(b)¶

Transform a binary sequence on the HC into an integer.

Parameters:
  • b: bool vector representing a genotype
Returns:
  • gt: integer representing the same genotype

Examples:

In [1]: integerify([False, True, True])
Out[1]: 3
